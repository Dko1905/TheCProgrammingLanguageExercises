# TheCProgrammingLanguageExercises
![C CI](https://github.com/Dko1905/TheCProgrammingLanguageExercises/workflows/C%20CI/badge.svg)<br/>
My own answers to the exercises in the book.

## Compiling
To compile all run
```sh
$ make
```
To compile one run
```sh
$ make 1.13
```
To clean
```sh
$ make clean
```
